"Функции в С++" 

main - главный и обязательный 

Функция - это именованная последовательность описаний операторов, выполняющая какое-либо законченное действие. Функция может примнимать параметры и возвращает значение

#include <iostream>

using namespace std; 

int sum(int x, int y)
{
    return x + y
}

int main() 
{
    int a = 5, b = 3, c;
    c = sum(a,b);
    cout << "sum = " << c << endl;
    cout << "sum = " << sum(a, b) << endl; 
    return 0;
}

Все величины, которые записываются внутри функции называются ЛОКАЛЬНЫМИ 

Глобальные расписываются вне функции main 

Механизм параметров является основным спобос обмена информацией между вызываемой и вызывающей функциями.

область действия - это честь программы, в которой переменную можно использовать для доступа к связанной с ней области памяти.


#include <iostream>

using namespace std; 

int main()
{
    int a = 5, b = 4, P, S;
    P = a + a + b + b
    S = a * b 
    cout << "P = " << P << endl; 
    cout << "S = " << S << endl;
    return 0; 
}
